programming | 环境配置

2023-04-04 4 min read # programming

在跑项目玩代码之前需要先自己创建一个折腾的环境，自己最开始是使用virtualenv配置虚拟环境（linux里我用的多），后来逐渐转为使用anaconda（windows里我用的多）。下面总结一些总是用到但容易忘的配置步骤。

Anaconda相关

win10 conda安装

step1

参考https://zhuanlan.zhihu.com/p/61639212，但注意测试是否安装成功是在Anaconda prompt （Anaconda Prompt is a command line interface with Anaconda Distribution. 直接搜这个终端 ）

step2

到这里只能说明conda安装完毕，但是环境变量还需要配置三个

参考   https://blog.csdn.net/moshiyaofei/article/details/103317490#:~:text=%E4%B8%80%E3%80%81%E7%AC%AC%E4%B8%80%E7%A7%8D%E6%96%B9%E6%B3%95,%E7%82%B9%E5%87%BB%E7%A1%AE%E5%AE%9A%E5%B0%B1%E5%8F%AF%E4%BB%A5%E4%BA%86%E3%80%82

C:\Users\xx\Anaconda3\Scripts
C:\Users\xx\Anaconda3\Library\bin
C:\Users\xx\Anaconda3\Library\mingw-w64\bin

然后在cmd就可以查看版本了 (conda -V)

step3

# 创建环境
conda create --name py35 python=3.5

# 安装好后，使用activate激活某个环境
activate py35 # for Windows
（注意：在git bash中，激活需要用source activate xx）

# 如果想返回默认的python 2.7环境，运行
deactivate py35 # for Windows

# 删除一个已有的环境
conda remove --name py35 --all

# 查看当前系统中已经安装了哪些环境
conda info -e

conda安装第三方库

# 为当前环境安装库，例如 numpy
conda install numpy
# conda会从从远程搜索numpy的相关信息和依赖项目

# 查看已经安装的packages
conda list

# 查看某个指定环境的已安装包
conda list -n py35

conda安装pytorch

参考pytorch官网https://pytorch.org/
注意要通过nvidia-smi确定cuda版本号
python版本也有要求（安装的torchvision要与python兼容）

virtualenv相关

参考https://www.cnblogs.com/shenjianping/p/10973197.html

虚拟环境配置步骤

1.安装virtualenv（如果server上没有提前安装）（需要关闭代理）

pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple virtualenv

2.创建虚拟环境

virtualenv -p /usr/bin/python3 ''环境名''
或
virtualenv -p python3 ''环境名''
# 不指定版本则使用系统默认版本

3.激活虚拟环境

source venv/bin/activate#进入虚拟环境  
# Windows 系统下运行 source venv/Scripts/activate

4.退出虚拟环境

deactivate

virtualenv安装pytorch

注意：python,torch,torchvision,CUDA版本匹配（CUDA是用于GPU的，如果linux上没有GPU，则安装CPU版本的torch）

查看linux CUDA版本：

nvcc -V

具体过程：
1.创建虚拟环境

mkvirtualenv --python=python3.6 MLMAN1

2.在官网上（https://pytorch.org/）找到对应CUDA=9.0的版本连接
3.linux上下载安装（下面这是一行）

pip install torch===1.0.0 torchvision===0.2.0 -f https://download.pytorch.org/whl/cu90/torch_stable.html

备注：
查看该版本torch支持的cuda版本

python
>>>import torch
>>>torch.version.cuda
查看torch版本
torch.__version__
查看cuda是否可用
torch.cuda.is_available()
